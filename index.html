Estrutura básica de conteúdo HTML



Subtítulo nível 2: O uso de parágrafos

O parágrafo é escrito através da tag <p> e terminada com a tag </p>.



Subtítulo nível 3: Formatação de texto

Para formatar o texto em negrito utiliza-se a tag <strong> e </strong>.


Para formatar o texto em itálico utiliza-se a tag <em> e </em>.


Para formatar o texto em sublinhado utiliza-se a tag <u> e </u>.



Subtítulo nível 4: Quebras de linha

Para criar uma quebra de linha num parágrafo ou entre os próprios elementos do código HTML se utiliza a tag <br>.



Subtítulo nível 5: Criação de link

Para gerar links utiliza-se a tag <a href="link" target="blank">: por exemplo, o link do Google: Google.
